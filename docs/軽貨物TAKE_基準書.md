# 軽貨物TAKE 基準書

## 1. サイトの目的
軽貨物TAKEは、山口県下関市を中心にスタートする軽貨物サービスの受付サイトである。
最初の目的は以下の2つに絞る。

1. 配送相談を受ける
2. 協力ドライバー登録を受ける

## 2. エリア方針
- 下関市中心で始める
- 将来的に山口県内へ広げる前提
- 最初から全国対応を強く打ち出さない

## 3. 公開方針
- 顧客名簿は公開しない
- ドライバー名簿は公開しない
- ドライバーの顔写真、プロフィール一覧は出さない
- 運営者個人の顔写真、住所、個人メールなどの個人情報は出さない
- サービス名「軽貨物TAKE」を前面に出して信頼を作る

## 4. 初期運用方針
- ドライバー登録は基本ウェルカム
- 最初は登録無料
- 条件が合う案件のみ個別連絡する
- 情報は裏側で管理する
- 表ではシンプル、裏では名簿蓄積できる構造にする

## 5. 連絡導線
最初の連絡導線は以下を基本とする。

- ホームページのフォーム送信
- Googleスプレッドシートへ自動保存
- 自動返信メール送信

LINEは最初の入口にしない。
LINEは将来的に通知や任意連絡で追加できる構造にする。

## 6. 初期公開範囲
最初に公開するのは以下の3つだけ。

1. トップページ
2. 配送相談フォーム
3. ドライバー登録フォーム

## 7. 今は出さないもの
- 車両販売
- 車両修理
- 車検
- 開業相談
- 事業相談
- ドライバー紹介
- 運営者プロフィール
- 細かい料金表
- 全国対応の強い打ち出し

## 8. トップページの優先順位
1. 配送を頼みたい人
2. 協力ドライバー登録
3. 地元企業にも使える印象づくり
4. 軽貨物TAKEという名前を覚えてもらう

## 9. デザイン方針
- 白ベース
- 青系中心
- 清潔感重視
- 高齢者でも読みやすい
- ボタンは大きく押しやすい
- 難しい言葉を使わない
- 地元感、相談しやすさ、小回りの利く印象を出す

## 10. 将来拡張
将来的には以下へ広げられる構造にする。

- LINE通知
- 品質評価
- 案件管理
- ドライバー選別
- 山口県内拡大
- アプリ化
