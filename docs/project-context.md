# 軽貨物TAKE プロジェクト前提

## サイト名
軽貨物TAKE

## 目的
1. 配送相談を受ける
2. 協力ドライバー登録を受ける

## エリア方針
- 下関市中心で開始
- 将来的に山口県内へ拡大
- 最初から全国対応を強く打ち出さない

## 公開方針
- 初期公開範囲は以下の3つ
  1. トップページ
  2. 配送相談フォーム
  3. ドライバー登録フォーム
- 顔出ししない
- 公開プロフィール一覧は作らない
- 個人情報、運営者プロフィール、顔写真、個人メールは出さない

## 受付方針
- フォーム送信が正本
- メールが基本
- LINEは補助導線
- 顧客もドライバーも裏側で名簿管理
- 初期は登録無料

## 技術構成
- フロント: React + Vite
- バックエンド: Google Apps Script
- 保存先: Googleスプレッドシート
- 通知: メール + LINE Messaging API
- LINE Notify は使わない

## 運賃計算器
- 名称は「運賃計算器」
- 見積額はリアルタイム更新
- 0円固定は不可
- 入力内容は保持する
- 戻っても入力は消さない
- 郵便番号と住所は必須
- 荷物内容や前画面情報は引き継ぐ

## ボタン文言
- この内容で見積依頼する
- この内容で依頼する

## 見積依頼 / 正式依頼フォーム
### 必須
- 名前
- メールアドレス
- 電話番号
- 郵便番号
- 住所

### 引き継ぎ項目
- 集荷先
- 納品先
- 荷物内容
- 希望日時
- 備考
- 計算結果
- 受付種別

## 同意文
### 見積依頼
入力内容の確認およびご連絡のため、個人情報の取り扱いに同意します

### 正式依頼
入力内容をもとに依頼受付を行うこと、および確認連絡のための個人情報の取り扱いに同意します

## 送信時の処理
1. Google Sheets 保存
2. 管理者へメール通知
3. 管理者へLINE通知
4. ユーザーへ受付確認メール

## 保存・通知ルール
- 保存最優先
- 保存に成功したら通知失敗でも受付成功扱い
- エラーや通知失敗はログに残す
